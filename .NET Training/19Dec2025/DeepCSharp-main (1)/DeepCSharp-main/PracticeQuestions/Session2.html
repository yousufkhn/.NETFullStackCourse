C#.NET Programming: Control Flow Notes

Control flow structures are the "brain" of your code, determining which blocks of code execute based on logic and repetition.

1. Conditional Statements

Conditional statements allow the program to make decisions and branch the execution path.

Syntax Overview

// If-Else-If Ladder
if (condition) { 
    // code 
} else if (otherCondition) { 
    // code 
} else { 
    // code 
}

// Switch Case
switch (variable) {
    case value1: 
        // code 
        break;
    default: 
        // code 
        break;
}

15 Critical Coding Exercises: Conditionals

  1. Height Category: Accept height in cm. If < 150 (Dwarf), 150-165 (Average), 165-190 (Tall), >190 (Abnormal).
  2. Largest of Three: Take three integers and find the maximum using nested if.
  3. Leap Year Checker: Determine if a year is leap (Divisible by 400 OR (Divisible by 4 and NOT 100)).
  4. Quadratic Equation: Calculate roots of $ax^2 + bx + c = 0$ using if-else to check the discriminant.
  5. Admission Eligibility: Math ≥ 65, Phys ≥ 55, Chem ≥ 50 AND (Total ≥ 180 OR Math+Phys ≥ 140).
  6. Electricity Bill: Calculate bill: First 199 units @ 1.20; 200-400 @ 1.50; 400-600 @ 1.80; above 600 @ 2.00. Add 15% surcharge if bill > 400.
  7. Vowel or Consonant: Use a switch statement to check if a character is a vowel.
  8. Triangle Type: Check if a triangle is Equilateral, Isosceles, or Scalene based on side lengths.
  9. Quadrant Finder: Take (x,y) coordinates and print which quadrant they lie in.
  10. Grade Description: Input grade (E, V, G, A, F) and print (Excellent, Very Good, Good, Average, Fail) using switch.
  11. Valid Date Check: Input day/month/year and check if it's a valid calendar date (handle Feb 29).
  12. ATM Withdrawal: Nested if to check: 1. Card inserted, 2. Pin valid, 3. Balance sufficient.
  13. Profit/Loss: Calculate profit or loss percentage based on Cost Price and Selling Price.
  14. Rock Paper Scissors: Logic for a 2-player game using nested conditionals.
  15. Simple Calculator: Use switch to perform +, -, *, / based on user operator input.

2. Loops and Jump Statements

Loops are used to repeat a block of code until a specific condition is met.

Syntax Overview

// For Loop (Known iterations)
for(int i=0; i<10; i++) { ... }

// While Loop (Condition based)
while(condition) { ... }

// Do-While (Runs at least once)
do { ... } while(condition);

// Jump Statements
break;    // Exit loop
continue; // Skip current iteration
goto Label; // Jump to specific line

15 Critical Coding Exercises: Loops

  1. Fibonacci Series: Display the first N terms of the Fibonacci sequence.
  2. Prime Number: Check if a number is prime using a for loop and break.
  3. Armstrong Number: Check if a number equals the sum of its digits raised to the power of number of digits.
  4. Reverse & Palindrome: Reverse an integer and check if it is a palindrome using while.
  5. GCD and LCM: Find the Greatest Common Divisor and Least Common Multiple of two numbers.
  6. Pascal's Triangle: Use Nested Loops to print Pascal's triangle up to N rows.
  7. Binary to Decimal: Convert a binary number string to decimal without using built-in library functions.
  8. Diamond Pattern: Print a diamond shape using * characters with nested loops.
  9. Factorial (Large numbers): Calculate N! and handle potential overflow for larger integers.
  10. Guessing Game: Use do-while to let a user guess a secret number until they get it right.
  11. Sum of Digits: Repeatedly sum digits of a number until the result is a single digit (Digital Root).
  12. Continue Usage: Print numbers from 1 to 50, but skip all multiples of 3 using continue.
  13. Menu System: Use do-while and switch to create a persistent console menu.
  14. Strong Number: Check if the sum of the factorial of digits is equal to the number itself.
  15. Search with Goto: Implement a deep-nested loop search that uses goto to exit all levels instantly upon finding a result.
Pro Tip: Avoid overusing goto in modern C# development as it can lead to "spaghetti code." Prefer break, return, or throw.
